# AutoDev-Tools-2026

Ferramenta educativa para análise de APKs Android, inspirada em ferramentas antigas como AutoDev Tools (2015), mas atualizada para fins modernos de aprendizado.

## Aviso

Este projeto é apenas para fins educacionais.
Use somente em aplicações próprias ou com permissão.

## Funcionalidades (em desenvolvimento)

- Análise de APK
- Leitura de AndroidManifest
- Visualização de recursos
- Descompilação de código
- Interface simples para desktop

## Objetivo

Aprender como aplicativos Android funcionam internamente, incluindo estrutura, permissões e código.
